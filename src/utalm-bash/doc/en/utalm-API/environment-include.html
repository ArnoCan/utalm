The following types of environments are provided by UTALM:
  • Runtime Environments for multiple OS
    Initally Linux versions only, but may work on any Unix like OS with GNU toolset.
  • Development Environments for Linux and GNU-Tools, others planned.

HowTo

  • Howto create and setup a command line utility project.
  • Howto create and setup an Eclipse project.

    1. Create a project in eclipse, e.g.: 
      ${MYWORKSPACE}/utalm-python-test
    2. Open a shell in the projects root directory and set INSTROOT, e.g.: 
      export INSTROOT=${MYWORKSPACE}/utalm-python-test
    3. Proceed with : Howto create and setup a command line utility project.
    4. Add specific Eclipse procedures :
      bash C++ Python
