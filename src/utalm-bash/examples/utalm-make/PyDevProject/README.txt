1. Create a project in eclipse, e.g.:
   ${MYWORKSPACE}/utalm-python-test

2. Open a shell in the projects root directory and set INSTROOT, e.g.:
   export INSTROOT=${MYWORKSPACE}/utalm-python-test

3. Install "utalm-bash" package, default in 
   "/usr/share/utalm-bash-<version>-<release>"
 
4. Install "utalm-bash" software by installer, e.g.:
   cd /usr/share/utalm-bash-<version>-<release>
   . sourceEnvironment.sh
   ./install.sh
